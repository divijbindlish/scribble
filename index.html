Welcome to Awesome Notes

Awesome Notes is a simple app which allows you, the user to create and manage different lists. These lists may contain any information that the user desires to store.

The innovative features of Awesome Notes:

  • Includes a checkbox for all tasks that allow user to cross out all completed tasks
  • Allows the user to add and remove tasks at will
  • Built using ajax - that means no more stupid page reloads.
